## 📌 Tentang Katana Chronicles
Katana Chronicles adalah ensiklopedia digital interaktif khusus yang membahas seluk-beluk pedang tradisional Jepang, Katana. Proyek ini memadukan fakta sejarah orisinal dengan representasi budaya populer, menjadikannya menarik bagi pencinta sejarah sekaligus penggemar anime.

### ✨ Fitur Utama
* **Historical Timeline:** Menjelajahi evolusi pedang Jepang dari zaman Kamakura hingga era modern secara interaktif.
* **Craftsmanship Guide:** Penjelasan visual mengenai tahapan pembuatan Katana (dari pemilihan besi *Tamahagane* hingga proses pelipatan baja).
* **Anime Connection Hub:** Komparasi menarik antara pedang legendaris di dunia nyata dengan pedang ikonik di anime terkenal (seperti Demon Slayer, One Piece, atau Bleach).
* **Interactive Gallery:** Galeri gambar berkualitas tinggi dengan deskripsi detail mengenai anatomi pedang (*Tsuba, Habaki, Saya*, dll).

### 🛠️ Tech Stack
* **Core:** HTML5, CSS Carousel & Overlay effects, JavaScript (Interactive gallery & tabs controller).
